Data Profile

Information about the training data used in our pet classification model.

Dataset Statistics

  • Total number of images: 10,000+
  • Number of cat breeds: 42
  • Number of big cat species: 15
  • Image resolution range: 224x224 to 1024x1024 pixels
  • Dataset collection period: 2020-2024

Bias Assessment

Analysis of potential biases in our classification model.

Known Biases

  • Higher accuracy for commonly photographed breeds
  • May perform better on professional photos than casual snapshots
  • Limited performance on mixed-breed cats
  • Potential confusion between visually similar breeds

Mitigation Strategies

We continuously work to improve our model by:

  • Expanding our dataset with diverse images
  • Implementing data augmentation techniques
  • Regular model evaluations across breed categories
  • Collecting user feedback to identify misclassifications
